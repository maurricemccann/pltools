# Psyched Luxuries

Four instruments, one practice. Installable as a Progressive Web App on iOS, Android, and desktop.

Live build: deployed via GitHub Pages.

## Local testing

    python server.py

## Files

- `index.html` — launcher
- `codex.html`, `drops.html`, `outreach.html`, `seeds.html` — sub-apps
- `manifest.webmanifest`, `sw.js` — PWA core
- `icons/` — apple-touch-icon, PWA icons, tile glyphs
- `.nojekyll` — disables GitHub's Jekyll preprocessor
